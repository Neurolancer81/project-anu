# ProjectAnu Documentation

This folder contains the Game Design Document (GDD) for ProjectAnu, a turn-based tactics game inspired by XCOM.

## Opening the GDD

This documentation is organized as an **Obsidian vault** for easy navigation and linking between design documents.

### Setup Obsidian
1. Download Obsidian from https://obsidian.md
2. Open Obsidian
3. Click "Open folder as vault"
4. Select this `documentation` folder
5. Start with `GDD/GDD-Structure.md`

### Folder Structure

```
documentation/
├── .obsidian/          # Obsidian configuration
└── GDD/                # Game Design Document
    ├── 00-Overview/    # High-level vision and concept
    ├── 01-Core-Mechanics/  # Fundamental gameplay systems
    ├── 02-Systems/     # Supporting technical systems
    ├── 03-Meta-Layer/  # Campaign and base management
    ├── 04-Content/     # Specific content catalogs
    ├── 05-UI-UX/       # Interface and experience design
    ├── 06-Technical/   # Architecture and requirements
    └── 07-Development/ # Project planning and milestones
```

## Quick Start

- **Main Index:** `GDD/GDD-Structure.md`
- **Game Vision:** `GDD/00-Overview/Game-Vision.md`
- **Development Plan:** `GDD/07-Development/Project-Milestones.md`

## Viewing Without Obsidian

All files are standard Markdown and can be viewed in any text editor or Markdown viewer. However, internal `[[links]]` work best in Obsidian.
